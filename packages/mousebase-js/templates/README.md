# MouseBase Templates

Ready-to-run starter templates for building AI agents with MouseBase.

| Template | Description |
|----------|-------------|
| [nextjs-chatbot](./nextjs-chatbot/) | Full-stack chatbot UI with streaming responses using Next.js |
| [express-agent](./express-agent/) | Express API server wrapping a MouseBase agent |
| [rag-app](./rag-app/) | Retrieval-Augmented Generation app with document ingestion |
| [discord-bot](./discord-bot/) | Discord bot powered by a MouseBase agent |
| [customer-support](./customer-support/) | Multi-agent customer support system with escalation |
| [research-assistant](./research-assistant/) | Web research agent with search and summarization |

## Quick Start

```bash
export MOUSEBASE_API_KEY=sk-...
cd <template-name>
npm install
npm run dev
```
